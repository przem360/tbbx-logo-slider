# tbbx-logo-slider
Simple image ticker for wordpress

## usage
1. Download, install and activate plugin in WordPress
2. Go to "Assosiates logos" (WordPress main admin menu)
3. Add images as CPT
4. Add [tls-shortcode] to page or post
